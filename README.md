# lfs-vcpkg
Linux From Scratch ports for VCPKG

LFS Version: https://www.linuxfromscratch.org/lfs/view/12.4-systemd/

Pre reqs Ubuntu 24.04:
```
sudo apt-get install build-essential git curl zip unzip tar cmake ninja-build pkg-config autoconf autoconf-archive automake libtool libtool-bin m4 tree
```

Build:
```
chmod a+x ./build.sh
./build.sh

```
